# 🦅 Onboarding Eagle 2.5

This guide provides step-by-step instructions from environment setup to training, evaluation, and launching a Streamlit demo.

---

### 📋 Main Steps

```
# Clone the repository
git clone git@github.com:cg1177/eagle-video.git eagle

# Switch to the target branch
cd eagle
git checkout eagle2.5

# Navigate to the working directory
cd Eagle2

```
> ⚠️ Note: All commands below should be executed within the eagle/Eagle2 directory.



1. ⚙️ [Setting up `.env` file](./1.setting_env_file.md)  
   Configure the environment variables required to run Eagle.

2. 🚀 [Launching Jupyter Notebook](./2.start_a_jupyter.md)  
   Start a Jupyter environment for interactive development.

3. 🦅 [Installing Eagle](./3.installing.md)  
   Install Eagle and all necessary dependencies.

4. 📊 [Preparing Data](./4.preparing_playground.md)  
   Prepare your dataset in the required format.

5. 💪 [Starting Training](./5.training.md)  
   Train the Eagle model using your prepared data.

6. ✅ [Running Evaluation](./6.evaluation.md)  
   Evaluate your trained model’s performance.

7. ✨ [Launching Streamlit Demo](./7.streamlit_demo.md)  
   Run an interactive Streamlit demo to visualize results.

8. 🔮 [Model Inference](./8.inference.md)  
   Perform inference using the trained model.
